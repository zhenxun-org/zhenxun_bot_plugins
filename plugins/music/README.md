# 点歌插件

用于在群聊/私聊里按歌名点歌，当前数据源为网易云音乐。

## 功能

- 按歌名搜索歌曲
- 发送音乐卡片（普通模式）
- 发送真寻风格图片卡片 + 播放链接（`zhenxun` 模式）

## 指令

- `点歌 [歌名]`
  - 示例：`点歌 晴天`
  - 示例：`点歌 反方向的钟`

不带歌名时，插件会继续询问你输入歌名。

## 配置项

- `music.type`（默认：`zhenxun`）
  - `zhenxun`：发送图片卡片 + 音乐链接
  - `normal`：发送平台音乐卡片（OneBot music segment）

## 说明

- 搜索结果默认取第一首匹配歌曲。
- 当 `zhenxun` 模式渲染失败时，会自动回退到 `normal` 模式发送。

## 常见问题

- 提示“没有找到这首歌”
  - 可能是歌名不完整，建议加上歌手或更准确的关键词。
- 提示“点歌查询失败，请稍后再试”
  - 多为上游接口波动，稍后重试即可。
