# 钉宫骂我（dinggong）

随机发送一段“钉宫系”语音，并可附带对应文本。

## 使用方法

本插件需要“对机器人说”才会触发（`to_me` 规则）。

常用触发方式：

```bash
@机器人 骂我
@机器人 骂老子
真寻骂我
小真寻骂我
```

说明：

- 指令内部名是 `ma-wo`，日常直接用上面的自然语言即可。
- 有 3 秒冷却，连续触发会提示：
  - `就...就算求我骂你也得慢慢来...`

## 资源文件

语音资源目录：

```text
resources/record/dinggong
```

目录中应包含：

- 音频文件（示例：`01.mp3`、`02.mp3`）
- `data.json`（可选文本映射）

插件安装时会把插件目录下的 `dinggong/` 自动移动到上面的资源目录。

## 自定义语音包（可选）

1. 把音频放进 `resources/record/dinggong`（建议使用数字文件名，如 `61.mp3`）。
2. 在 `data.json` 里增加同编号文本（键名不带扩展名）。

示例：

```json
{
  "61": "笨蛋！这种事情也要我教吗！"
}
```

触发时会随机选一个音频发送；如果 `data.json` 有对应键，会再发送一条文本。
