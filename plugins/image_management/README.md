# 图库操作插件

一个用于管理本地图库的插件，支持发图、上传、删除、移动图片。

## 功能说明

- 公开图库发图（随机或按 id 指定）
- 查看公开图库列表
- 管理员上传图片（单张/连续）
- 管理员删除图片
- 管理员在图库间移动图片

## 普通用户用法

- 发送：`[图库名]`
  - 示例：`美图`
- 发送：`[图库名] [id]`
  - 示例：`萝莉 2`
- 查看公开图库：`查看公开图库`

## 管理命令（需要 @机器人）

- `上传图片 [图库] [图片]`
  - 示例：`@机器人 上传图片 美图 [图片]`
- `连续上传图片 [图库]`
  - 发送多张图片，最后发送 `stop` 结束
- `删除图片 [图库] [id]`
  - 示例：`@机器人 删除图片 美图 12`
- `移动图片 [源图库] [目标图库] [id]`
  - 示例：`@机器人 移动图片 萝莉 美图 6`

说明：管理员命令中的“图库”既可输入名称，也可输入 `查看公开图库` 显示的序号。

## 配置项

- `IMAGE_DIR_LIST`：公开图库名称列表，默认 `['美图', '萝莉', '壁纸']`
- `WITHDRAW_IMAGE_MESSAGE`：自动撤回配置，格式 `(延迟秒, 聊天类型)`
  - 聊天类型：`0` 私聊，`1` 群聊，`2` 群聊+私聊
- `SHOW_ID`：发图时是否显示图片 id，默认 `True`
- `UPLOAD_IMAGE_LEVEL`：上传图片所需管理员等级，默认 `6`
- `DELETE_IMAGE_LEVEL`：删除图片所需管理员等级，默认 `7`
- `MOVE_IMAGE_LEVEL`：移动图片所需管理员等级，默认 `7`

## 注意事项

- 插件按 `jpg` 文件进行管理。
- 删除/移动后会自动整理 id（保持编号连续）。
- 如果发送提示“该 id 不存在”，请先不带 id 发送一次，或使用 `查看公开图库` 后按范围输入。
