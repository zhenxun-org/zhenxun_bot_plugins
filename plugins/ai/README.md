# AI 插件使用说明

这个插件的作用很简单：和机器人聊天。

## 一句话概括

在群里 `@机器人` 说话，或者在私聊里直接发消息，它就会回你。

## 怎么用

1. 在群聊里先 `@机器人`。
2. 发送你想说的话，比如：
   `@真寻 今天天气怎么样`
   `@真寻 讲个笑话`
   `@真寻 你在吗`

## 它会怎么回复

- 你打招呼（如“你好”“在吗”）时，通常会回一张图 + 一句问候。
- 正常聊天时，会给出文字回复。
- 偶尔会有一些更“活泼”的随机回复。
- 如果它没听懂，会给你一个“没听懂”的提示。

## 使用小贴士

- 问题尽量具体一点，回复通常会更有用。
- 连续发完全一样的话，或反复学它说话，可能会触发“生气”机制。
- 触发后有概率被临时禁言一小段时间（通常几十秒到几分钟）。

## 常见问题

### 为什么我发了消息它不回？

最常见原因是群里没有 `@机器人`。  
先 `@` 再说话，一般就能触发。

### 为什么有些词被替换成 `*`？

这是机器人做了文本过滤，属于正常行为。
