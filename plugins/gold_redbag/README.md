# 金币红包插件

一个群聊里的“发红包 / 抢红包”小游戏插件，支持普通红包和节日红包。

## 这个插件能做什么

- 发红包：把自己的金币拆成多个红包发到群里
- 抢红包：群成员随机领取金币
- 退回红包：发红包后在冷却时间后可退回未领取部分
- 节日红包：超级用户可一键给多个群发节日红包

## 普通用户命令

- `塞红包 [金币数] ?[红包数=5] ?[at指定人]`
  - 示例：`塞红包 1000`
  - 示例：`塞红包 1000 10`
  - 示例：`塞红包 500 @小明`（指定人红包，数量会自动变为 1）
- `开` / `抢` / `开红包` / `抢红包`
  - 领取当前群里可领取的红包
- `退回红包` / `退还红包`
  - 退回自己未领完的红包（需等待冷却时间）

## 超级用户命令

- `节日红包 [金额] [红包数] ?[主题文字] ? -g [群id] [群id] ...`
  - 示例：`节日红包 10000 20 新年快乐`
  - 示例：`节日红包 10000 20 活动福利 -g 123456789 987654321`
  - 不带 `-g` 时会尝试发到可获取到的群列表

## 使用规则

- 金额必须大于 0，且不能小于红包数量
- 金额不足时无法发送
- 指定人红包只有被指定用户可领取
- 同一次节日红包活动中，用户在不同群只能领取 1 次

## 配置项

- `DEFAULT_TIMEOUT`：普通红包过期时间（秒），默认 `600`
- `DEFAULT_INTERVAL`：用户发送红包的最小间隔（秒），默认 `60`
- `RANK_NUM`：结算排行榜展示人数，默认 `10`

## 常见提示

- `没有红包给你开`：当前没有你可领取的红包
- `目前没有红包可以退回`：你当前没有可退回的红包
- `你的红包还没有过时`：还没到可退回时间
