# 刷屏禁言插件

用于检测群聊中的重复刷屏行为，并自动禁言。

## 功能说明

- 监听群消息（文本与图片）
- 在指定时间内统计同一用户的重复消息次数
- 超过阈值后自动禁言指定分钟数
- 支持按群单独配置

## 指令

- `刷屏设置`
  - 查看当前群配置
- `刷屏设置 -c [次数]`
  - 设置检测次数阈值
- `刷屏设置 -t [秒数]`
  - 设置检测时间窗口
- `刷屏设置 -d [分钟]`
  - 设置禁言时长

示例：

- `刷屏设置 -c 10`
- `刷屏设置 -t 7 -d 10`
- `刷屏设置 -d 0`（关闭本群刷屏禁言）

## 规则

当用户在 `T` 秒内发送相同消息超过 `N` 次时，执行禁言 `M` 分钟。

- `T` = `-t` 设置值
- `N` = `-c` 设置值
- `M` = `-d` 设置值

## 配置项

- `MUTE_LEVEL`：修改刷屏设置所需管理员等级（默认 `5`）
- `MUTE_DEFAULT_COUNT`：默认检测次数（默认 `10`）
- `MUTE_DEFAULT_TIME`：默认检测时间窗口，单位秒（默认 `7`）
- `MUTE_DEFAULT_DURATION`：默认禁言时长，单位分钟（默认 `10`）

## 注意事项

- 仅在群聊生效，私聊不生效。
- 机器人需要有群管理权限才能执行禁言。
- 禁言时长设置为 `0` 可关闭当前群的刷屏禁言功能。
- 目前按“消息内容一致”判断重复（文本 + 图片哈希）。
