# Epic 免费游戏（epic）

查询 Epic 商店当前（或即将）免费的游戏信息。

## 使用方法

发送：

```bash
epic
```

## 返回内容

通常会包含：

- 游戏封面图
- 游戏名与原价
- 简介
- 开发商 / 发行商
- 免费截止时间（UTC）
- 商店链接

在群聊中，支持的平台会使用“合并转发”发送多条游戏信息。

## 常见问题

- 提示 `Epic 可能又抽风啦，请稍后再试（`
  - 一般是 Epic 接口临时异常或网络问题，稍后重试即可。

- 游戏链接打开 404
  - 少数游戏的返回数据不完整，拼接链接可能失效，通常过一段时间会恢复。
